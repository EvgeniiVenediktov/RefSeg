In [*] was introduced a Query Generation Module (QGM), which generates query vectors from the input language and image features, representing different possible interpretations of the language description. These queries allow the model to explore various ways of understanding the target object. The Query Balance Module (QBM) acts as a filter that assigns confidence scores to the multiple queries. This lightweight feed-forward network selectively fuses the query responses based on these confidence scores, ensuring that only the most relevant interpretations contribute to the final segmentation mask. These two modules work together to prominently boost the diversity to understand the image and query language, and enhance the model’s robustness towards highly stochastic inputs. To fuse extracted language and image features was used a Spatial Dynamic Multi-Modal Fusion module. It dynamically adjusts the language features for each image region using an attention-based mechanism to compute spatial correlations. This approach allows the model to assign tailored linguistic information to different parts of the image based on context. The VLT framework achieved state-of-the-art results on several referring image segmentation datasets, such as RefCOCO, RefCOCO+, and G-Ref.
